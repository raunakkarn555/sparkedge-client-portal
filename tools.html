04 How to Add a New Link
➕ Adding a new tool card — 4 steps
Open tools.html in VS Code, find the right grid section, duplicate a card block and fill in your details. Done in under 2 minutes.
01
Open tools.html in VS Code
Find the section where you want to add — Client tools, Admin tools, or Guides.
02
Copy a <a> card block
Select any existing <a href="..." class="tool-card card-orange"> block and duplicate it.
03
Fill in 4 things
Change: href (URL), icon emoji, card-title, card-desc, and card-url text.
04
Push to GitHub
In VS Code terminal: git add .
git commit -m "add [tool name]"
git push Live in 60 seconds.
Copy this template
