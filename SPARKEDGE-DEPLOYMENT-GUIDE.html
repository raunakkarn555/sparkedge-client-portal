8
Useful Firebase Console Actions
👁
See all your client data live
Go to console.firebase.google.com → your project → Firestore Database → clients collection. You can see every ticket, edit fields directly, even from your phone.
👤
Add or remove team members
Firebase Console → Authentication → Users → Add User. Give them email + password. They can log in to admin.html from anywhere. No file editing. To remove: delete from Firebase Console — they're instantly locked out.
🔑
Change your admin password
Firebase Console → Authentication → Users → find your email → click the 3-dot menu → Reset password. Or change it from Account Settings in Firebase.
💾
Backup your data anytime
In admin.html → click ⬇ Export button (top right) → downloads a complete JSON file with all tickets and vouchers. Save this to Google Drive weekly as a backup. It can be re-imported anytime.
